Constructum

Constructum is a custom-built CI/CD pipeline for my homelab.

It runs on top of Kubernetes, given that my homelab is based around my Kubernetes cluster.

Features

  • Runs simple pipelines defined in internal Git repositories
  • Has secret support to derive secrets from various secret stores

Upcoming Features

  • A user interface for easy access to logs / details
  • A command-line tool for running pipelines locally and communicating with the API server
  • Integration with the source-control tool I use
  • Multiple pipelines per repository
