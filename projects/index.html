  • Constructum
    Constructum is a custom-built CI/CD pipeline for my homelab. It runs on top of Kubernetes, given that my homelab is based around my Kubernetes cluster. Features Runs simple pipelines defined in internal Git repositories Has secret support to derive secrets from various secret stores Upcoming Features A user interface for easy access to logs / details A command-line tool for running pipelines locally and communicating with the API server Integration with the source-control tool I use Multiple pipelines per repository
  • Theta
    Theta is my toy programming language, with roots coming from Lox of Crafting Interpreters fame, Java, and Rust. It’s designed to be a starting place for further exploration of programming language concepts I didn’t learn in college. Features Bytecode compiler and interpreter for basic expressions Support for statements Block support Static typing Stack frames Function support Array support Upcoming Features Structure / Object support Interface or traits? Generics Unified heap (arena support / raw pointers in Rust) Garbage collection (Z1 / Shenanoah based) A simple standard library Optimization passes SSF, etc Debug tooling JIT compilation Hot code detection
