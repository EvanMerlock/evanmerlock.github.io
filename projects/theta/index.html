Theta

May 05 2023 11:23PM

Theta is my toy programming language, with roots coming from Lox of Crafting Interpreters fame, Java, and Rust. It’s designed to be a starting place for further exploration of programming language concepts I didn’t learn in college.

Features

  • Bytecode compiler and interpreter for basic expressions
  • Support for statements
  • Block support
  • Static typing
  • Stack frames
  • Function support
  • Array support

Upcoming Features

  • Structure / Object support
  • Interface or traits?
  • Generics
  • Unified heap (arena support / raw pointers in Rust)
  • Garbage collection (Z1 / Shenanoah based)
  • A simple standard library
  • Optimization passes
    • SSF, etc
  • Debug tooling
  • JIT compilation
    • Hot code detection
